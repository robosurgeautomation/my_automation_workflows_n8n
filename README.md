# Prospect Harvester - Lead Generation 
Automates organization lead generation using Google Maps data and contact enrichment.

Workflow
Input: Reads keywords from Google Sheets

Scraping: Extracts business data from Google Maps via Apify

Scoring: Calculates confidence scores (0-25 points)

Enrichment: Adds emails, phones, social media profiles

Output: Saves structured lead data to Google Sheets

Data Collected
Organization details (name, website, phone, address, category)

Confidence score

Contact information & social media

Date added

Integrations
Google Sheets (I/O)

Apify (scraping)

Make.com (orchestration)

Ideal for sales teams automating lead generation from business directories.
